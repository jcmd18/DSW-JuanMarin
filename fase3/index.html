Propósito del sitio

El propósito de este sitio es dinamizar el aprendizaje mediante el uso de una herramienta digital, un Objeto Virtual de Aprendizaje OVA. En esta tercera fase se describen los principales conceptos de las estructuras básicas de HTML, atributos y valores.

Que es lo principal de esta etapa?

En esta fase del proyecto, se utilizarán las CSS (Hojas de Estilo en Cascada) que corresponde al código que se utiliza para dar estilo a la página web.

El CSS (Cascading Style Sheets) u Hojas de estilo en cascada en español, es un lenguaje de hojas de estilo, es decir, que permite aplicar distintos formatos de manera selectiva a elementos en documentos HTML. Por ejemplo, permite cambiar el color de fondo de los elementos que conforman la página HTML, el tamaño y tipo de letra o su posición dentro de la página, es la manera de dinamizar y hacer atractivos los sitios web desde la perspectiva de los usuarios
